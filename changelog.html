v2.1.0 2026-04-30 Major
Added
Skills Library page
New dedicated page centralizing all Claude Code slash-command downloads in four sections: Figma Skills, Design System, Documentation, and Workflow. Skills with companion reference files or scripts are bundled as one-click install zips.
skills-library.html skills-downloads/
Added
figma-generate-library — 4-phase design system builder
New skill that scaffolds a professional-grade Figma design system from an existing codebase. Orchestrates 20–100+ use_figma calls across 4 phases: Discovery (scan codebase for tokens and components), Foundations (create variable collections), File Structure (set up Figma pages), and Components (build variants with auto-layout and variable bindings). Includes 7 reference docs and 9 reusable helper scripts.
.claude/commands/figma-generate-library.md .claude/figma-generate-library/
Added
figma-create-design-system-rules — project rules generator
New skill that analyzes a codebase and generates project-specific design system rules for Claude Code (CLAUDE.md), OpenAI Codex CLI (AGENTS.md), or Cursor (.mdc). Rules encode component location, styling approach, token usage, and Figma-to-code workflow.
.claude/commands/figma-create-design-system-rules.md
Added
mui-design — MUI design system foundations skill
New skill covering the full MUI v7 component catalog (39 components across 6 categories), theme token system, Figma→MUI token mapping, accessibility rules, component design rules, and spec-writing conventions. Loaded automatically by audit-figma, document-feature-spec, and cross-product-check.

v2.0.0 2026-04-27 Major
Changed
Full site restructure — product-based architecture
Replaced the flat UX pattern category pages with a deep product/feature/platform tree. Site is now organised around 7 products (Biz, Workflows, CKID, Intra, Files, Updates, Goals) plus a Global section. Each product has a fixed set of top-level pages (Overview, Theme, Components, Layout, Glossary, Changelog) and nested feature folders containing platform-specific spec pages.
biz/ workflows/ ckid/ intra/ files/ updates/ goals/ global/
Added
Global section
Cross-product design system reference with four sub-sections: Foundations (10 pages — Color System, Typography, Spacing, Motion, Iconography, Accessibility, Error Patterns, Empty States, Loading States, Notification Patterns), Components (9 pages), Data (Entity Type), and Layout. Also includes Overview, Glossary, and Changelog.
global/_index.html global/foundations/ global/components/ global/data/ global/layout/ global/_glossary.html global/changelog.html
Added
Home prompt templates
Interactive prompt-builder on the home page. Eight templates for common documentation tasks: Add a new product, Add a new feature to a product, Add a platform delta to a feature, Audit for inconsistency, Audit Figma file against feature spec, Check cross-product consistency, Promote a component to product level, Promote a component to global level. Each template uses chip-based placeholder inputs with live substitution into a copyable prompt.
