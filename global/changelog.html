Global — Changelog

Version history for shared Foundations and Components. Tracks design token updates, new components, and cross-product changes.

v0.3.0 2026-04-29 Minor
Foundations
Changed
All 6 foundation pages fully authored
Color System, Typography, Spacing, Motion, Iconography, and Accessibility pages replaced with complete, content-rich guides. Each page includes MUI token references, usage rules, visual examples, and do/don't guidance. Companion .md files optimized for AI reference.
Data
Changed
Entity Type — group hierarchy and permission model
Entity Type page updated with group type definitions (Department: exactly one per user; User Group: zero or more), permission precedence diagram (Everyone < Group < Individual User), and a worked example showing conflict resolution.
Global Styling
Changed
Page description style updated across entire library
All pages now use a plain muted description paragraph instead of the blue blockquote alert box. The divider between title and description was also removed. Code snippets in descriptions now render with the blue monospace style consistent with body content.
Components
Changed
Component library completely rewritten — 39 MUI-based components
Replaced 9 legacy placeholder components (Button, Input, Modal, Table, Nav Shell, Toast, Pagination, Data Viz, Form Validation) with 39 fully documented MUI components organized into 6 categories: Inputs (12), Data Display (7), Feedback (5), Surfaces (3), Navigation (7), Utilities (5). Each component includes variants, sizes, states, colors, key options, and common use cases. Designer-focused — no implementation code.
