Typography

MUI's typography token system defines the type scale for all Colorkrew products. Inter is the default font for English and Latin-alphabet content. Noto Sans JP is used for Japanese text.

Font Families

All Colorkrew products must use the following fonts. Never substitute or introduce additional typefaces without approval.

English / Latin / Numbers
Inter
Inter — default for all UI text
Japanese
Noto Sans JP
Noto Sans JP — Japanese text only
  • Inter is the default for all UI text including English, other Latin-alphabet languages, and numerals.
  • Noto Sans JP is applied scoped to Japanese content via the CSS lang attribute or a wrapper class.
  • Never mix the two fonts within a single text element.
  • Load fonts from Google Fonts or host them locally for performance.

Token Format

typography.<variant> — e.g., typography.body1, typography.h4

Each variant carries: fontFamily, fontSize, fontWeight, lineHeight, and letterSpacing.

Usage Guidelines

  • Use body1 or body2 for general content text. Never use heading variants (h1–h6) for non-heading content.
  • The button variant is uppercase by default (MUI default) — products may override to sentence case in their theme.
  • Use caption for helper text, timestamps, and metadata.
  • Use overline sparingly — category labels and section prefixes only.
  • Products may adjust variant font sizes but must document all overrides in _theme.md.

Do / Don't

Do
  • Use Inter for all English, Latin, and number text
  • Use Noto Sans JP scoped to Japanese content
  • Follow the MUI type scale for all text variants
  • Document any variant overrides in _theme.md
Don't
  • Introduce custom fonts not in the approved set
  • Use arbitrary font sizes outside the scale
  • Use heading variants for labels, captions, or UI metadata
  • Mix Inter and Noto Sans JP in a single text element
