# Breakpoints

> MUI five-breakpoint system for responsive layouts. Always mobile-first (min-width).

## Scale
- xs: 0px — small phones
- sm: 600px — large phones, small tablets
- md: 900px — tablets, small laptops
- lg: 1200px — desktops, wide laptops
- xl: 1536px — wide monitors

## Mobile-first rule
Document breakpoints as "from X up" using min-width. MUI helpers: breakpoints.up(), breakpoints.down(), breakpoints.between().

## Common Layout
- xs–sm: temporary drawer, full-width content, 4-col grid
- md: mini rail, content offset, 8-col grid
- lg–xl: expanded drawer, content offset, 12-col grid
