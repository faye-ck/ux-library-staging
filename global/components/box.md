# Box

> **Category:** Utilities. Generic layout container with full sx prop support.

## Key Props
- component: rendered HTML element (default: div)
- sx: theme-aware inline styles
- display: flex / grid / inline / block

## Use When
- Layout wrapper needing sx prop access
- No semantic MUI component fits

## Do Not Use For
- One-direction flex layouts → Stack
- 12-column responsive grids → Grid
- Elevated surfaces → Paper
