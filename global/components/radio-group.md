# Radio Group

> **Category:** Inputs. Single-selection control — only one option can be active at a time.

## Variants
- Default
- Controlled
- Custom icon

## Sizes
- Small
- Medium

## States
Unselected · Selected · Disabled (individual or group) · Error

## Colors
Primary · Secondary · Success · Error · Warning · Info

## Key Options
- Vertical (default) or Horizontal orientation
- Label placement: end (default), start, top, bottom
- Error state with helper text via FormControl
- Standalone radio or grouped via RadioGroup

## Use
- Exclusive option selection
- Settings with mutual choices
- Survey single-answer questions

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
