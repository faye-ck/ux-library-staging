# Accordion

> **Category:** Surfaces. Expandable/collapsible content sections for progressive disclosure.

## Variants
- Default
- Controlled (custom expand logic)

## Sizes
- Full-width (default)

## States
Collapsed · Expanded · Disabled

## Colors
Default (surface background)

## Key Options
- Custom expand icon
- Default expanded on load
- AccordionSummary — header with toggle
- AccordionDetails — body content
- AccordionActions — footer buttons
- unmountOnExit — removes content from DOM when collapsed (performance)

## Use
- FAQ sections
- Settings panels
- Collapsible content
- Progressive disclosure

- Styling and theme overrides: product `_theme.md`
