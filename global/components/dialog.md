# Dialog

> **Category:** Feedback. Modal overlay requiring user interaction before returning to main content.

## Variants
- Default (modal)
- Full-screen
- Alert dialog
- Scrollable

## Sizes
- xs · sm · md (default) · lg · xl · Full-width

## States
Open · Closed

## Colors
Default (surface background)

## Key Options
- Max-width — xs / sm / md / lg / xl
- Full-screen — takes over entire viewport
- Scroll — paper (dialog scrolls) or body (page scrolls)
- Draggable — via custom PaperComponent
- Slide transition
- Sub-components: DialogTitle, DialogContent, DialogContentText, DialogActions

## Use
- Confirmation prompts
- Form dialogs
- Detail views
- Critical decision points

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
