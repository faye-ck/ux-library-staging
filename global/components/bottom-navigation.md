# Bottom Navigation

> **Category:** Navigation. Mobile primary nav bar. Replaces drawer at xs–sm breakpoints.

## Item Count
3–5 items (2 items → use Tabs)

## States per action
Inactive · Active (primary.main) · Disabled

## Key Options
- showLabels: always show labels (recommended)
- value / onChange: controlled
- Position: fixed bottom

## Use
- Mobile Web primary navigation (below sm breakpoint)
- Mobile App primary sections

## Note
Not used on desktop — drawer replaces at md. Product label definitions: _layout.md.
