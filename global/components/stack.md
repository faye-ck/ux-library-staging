# Stack

> **Category:** Utilities. One-dimensional flex layout with theme-spaced gaps.

## Direction
row · row-reverse · column (default) · column-reverse

## Key Props
- direction: row / column
- spacing: gap in theme.spacing units
- divider: insert Divider between children
- useFlexGap: CSS gap (recommended v7)

## Use
- Form field stacking
- Button rows
- Icon + label pairs
- Action bars

## Do Not Use For
- Two-dimensional layouts → Grid
- Irregular column spans → Grid
