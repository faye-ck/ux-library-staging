# Switch

> **Category:** Inputs. Binary toggle with immediate effect — for settings that turn something on or off.

## Variants
- Default
- iOS-style
- Android-style

## Sizes
- Small
- Medium

## States
Off · On · Disabled

## Colors
Primary · Secondary · Warning · Default

## Key Options
- Label via FormControlLabel (placement: start / end / top / bottom)
- Group multiple switches with FormGroup
- Required state

## Use
- Enable / disable settings
- Feature flags
- Notification preferences
- On/off controls

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
