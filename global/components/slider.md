# Slider

> **Category:** Inputs. Range input for selecting a value or range along a continuous or discrete track.

## Variants
- Continuous
- Discrete (step-based)
- Range (two-thumb)

## Sizes
- Small
- Medium

## States
Default · Active (dragging) · Disabled

## Colors
Primary · Secondary

## Key Options
- Horizontal (default) or Vertical orientation
- Marks — auto-generated or custom with labels
- Value label: auto (hover), on (always), off
- Track: standard, inverted, or removed
- Minimum distance enforcement for range sliders

## Use
- Volume / brightness control
- Price range filter
- Rating input
- Continuous value adjustment

- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
