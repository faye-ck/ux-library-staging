# Drawer

> **Category:** Navigation. Side panel that slides in from the screen edge for navigation or supplementary content.

## Variants
- Temporary (toggle overlay)
- Persistent (pushes content)
- Mini variant (narrow + expanded)
- Permanent (always visible)

## Sizes
- Default 240px (customizable)

## States
Open · Closed

## Colors
Default (surface background)

## Key Options
- Anchor — left (default), right, top, bottom
- SwipeableDrawer — touch/swipe gesture on mobile
- keepMounted — keeps DOM alive when closed
- Clipped — sits below App Bar
- Mini variant — icon strip when closed

## Use
- Side navigation
- Mobile hamburger menu
- Filter panel
- Detail / settings panel

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
