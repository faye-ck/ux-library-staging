# Chip

> **Category:** Data Display. Compact interactive or informational element representing a tag, filter, or input value.

## Variants
- Filled (default)
- Outlined

## Sizes
- Small
- Medium

## States
Default · Clickable · Deletable · Disabled

## Colors
Default · Primary · Secondary · Error · Info · Success · Warning

## Key Options
- Avatar — user image inside chip
- Start icon — decorative icon before label
- Delete icon — custom close/delete icon
- Clickable — acts as button or link
- Deletable — shows delete icon

## Use
- Filter tags
- User-entered input values
- Selectable options
- Status indicators
- Removable labels

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
