# Select

> **Category:** Inputs. Dropdown for selecting one or multiple options from a predefined list.

## Variants
- Outlined (default)
- Filled
- Standard

## Sizes
- Small
- Medium

## States
Default · Open · Focused · Disabled · Error · Required

## Colors
Primary · Secondary

## Key Options
- Multiple selection with chip display
- Grouped options via subheader
- Placeholder when no selection
- Helper text and error message
- Auto-width or fixed-width
- Native select fallback for mobile

## Use
- Single option selection
- Multi-select with chips
- Filter dropdowns
- Mobile-optimized selection

- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
