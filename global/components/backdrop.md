# Backdrop

> **Category:** Feedback. Full-screen dimmed overlay indicating a blocking UI state such as loading.

## Variants
- Default (semi-transparent dark)

## Sizes
- Full viewport

## States
Open · Closed

## Colors
Customizable via sx

## Key Options
- Click-to-close handler
- Z-index control for stacking
- Nested content — spinner or loading indicator
- Fade transition

## Use
- Loading overlay
- Focus narrowing behind dialog
- Blocking action in progress

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
