# Table

> **Category:** Data Display. Structured data in rows and columns — supports sorting, selection, and pagination.

## Variants
- Default
- Dense (compact padding)

## Sizes
- Full-width (default)
- Responsive via TableContainer

## States
Default · Row selected · Row hover · Sorted column

## Colors
Primary (selected rows, sort indicators)

## Key Options
- Sticky header — stays visible on scroll
- Dense padding — reduced row height
- Sorting — via TableSortLabel on headers
- Row selection — checkbox per row
- Collapsible rows — expandable detail sections
- Column grouping — merged header cells
- Pagination integration

## Use
- Data tables
- Admin lists
- Report views
- Inventory / asset management

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
