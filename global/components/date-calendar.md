# Date Calendar

> **Category:** Utilities. Standalone inline calendar — always visible, no input field. (MUI X)

## Variants
- Day view (default)
- Month view
- Year view

## Sizes
- Fixed calendar grid (customizable)

## States
Default · Date selected · Date disabled · Today highlighted

## Colors
Primary (default)

## Key Options
- Fixed week rows — consistent grid height
- Week number display
- Show days outside current month
- Disable specific dates
- Custom day rendering (events, highlights)
- Years per row / months per row

## Use
- Embedded calendar widget
- Event / booking calendar
- Scheduling interface
- Inline date selection
