# Button Group

> **Category:** Inputs. Groups related buttons with shared borders and consistent styling.

## Variants
- Contained
- Outlined
- Text

## Sizes
- Small
- Medium
- Large

## States
Default · Disabled (individual or full group)

## Colors
Primary · Secondary

## Key Options
- Horizontal (default) or Vertical orientation
- Disabled elevation — flat look for contained
- Individual button loading states
- Split button — last button opens a menu

## Use
- Grouped related actions
- Segmented control
- View / mode switching
- Split button with dropdown

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
