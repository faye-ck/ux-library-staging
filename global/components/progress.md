# Progress

> **Category:** Feedback. Indicates loading or task completion — linear bar or circular spinner.

## Variants
- Linear
- Circular

## Sizes
- Circular: 40px default (customizable)
- Linear: full-width

## States
Indeterminate (unknown duration) · Determinate (0–100%) · Buffer (linear only)

## Colors
Primary · Secondary · Success · Inherit

## Key Options
- Determinate — specific progress value 0–100
- Indeterminate — animated loop for unknown wait
- Buffer — buffered vs loaded progress (linear)
- Percentage label overlay
- Circular track ring
- disableShrink — reduces CPU on long animations

## Use
- File upload progress
- Page / data loading
- Form submission feedback
- Media buffering

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
