# Icon Button

> **Category:** Inputs. Compact icon-only action. Always pair with Tooltip.

## Sizes
- Small (24px)
- Medium (40px)
- Large (48px)

## States
Default · Hover · Focused · Disabled

## Colors
Default · Primary · Secondary · Error · Warning · Info · Success

## Key Options
- Tooltip (required — always provide accessible label)
- Edge: start / end (flush alignment)
- disableRipple

## Use
- Close / dismiss (Dialog, Drawer, Snackbar)
- Toolbar actions (edit, delete, share)
- Toggle actions (bookmark, favorite)
- Navigation controls

## Rule
Always wrap with Tooltip. Icon alone is insufficient for accessibility.
