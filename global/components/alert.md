# Alert

> **Category:** Feedback. Inline message communicating system status — success, info, warning, or error.

## Variants
- Standard (default)
- Filled
- Outlined

## Sizes
- Adapts to content

## States
Static · Dismissible (with close button)

## Colors
Success · Info · Warning · Error

## Key Options
- Severity — sets icon + color (success/info/warning/error)
- AlertTitle — bold headline above message
- Action — custom button or link inside alert
- Custom icon or hidden icon
- Dismissible — close button via onClose

## Use
- Form feedback
- System status banners
- Validation messages
- Inline warnings

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
