# App Bar

> **Category:** Surfaces. Top navigation bar containing branding, page title, and global actions.

## Variants
- Default
- Dense (compact)
- Prominent (tall)
- Bottom App Bar

## Sizes
- 64px desktop · 56px mobile · 48px dense

## States
Static · Elevated (on scroll) · Hidden (scroll-to-hide)

## Colors
Primary (default) · Inherit · Transparent

## Key Options
- Position: fixed, sticky, static, absolute, relative
- Scroll behaviors: hide-on-scroll, elevate-on-scroll, back-to-top
- Dense — reduced height for desktop
- Search field integration
- Hamburger menu for mobile drawer
- enableColorOnDark — maintains color in dark mode

## Use
- Main site header
- Page title bar
- Top navigation with actions
- Mobile header with back button

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
