Icon Button

Compact icon-only action button. Always pair with a Tooltip — the icon alone must never be the only affordance for an action.

Inputs

Default · Primary · Secondary · Error · Warning · Info · Success

Key Options

  • Tooltip (required — always wrap with Tooltip for accessible label)
  • Edge prop: start / end (removes padding on that side for flush alignment)
  • disableRipple (for icon-in-icon-button in custom surfaces)

Common Use Cases

  • Close / dismiss actions (Dialog, Drawer, Snackbar)
  • Toolbar actions (edit, delete, share)
  • Toggle actions (bookmark, favorite, expand)
  • Navigation controls (back, forward, open menu)
Always pair with <Tooltip title="...">. Never use an IconButton without a descriptive tooltip — icon meaning alone is insufficient for accessibility.
