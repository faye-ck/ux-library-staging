# Paper

> **Category:** Surfaces. Flat elevated surface underlying most MUI containers. Base layer for Card, Dialog, Menu, Drawer.

## Elevation
0 (flat) · 1 (default) · 3 (menus) · 8 (dialogs) · 24 (maximum)

## Variants
elevation (shadow scale) · outlined (border, no shadow)

## Key Options
- elevation: 0–24
- variant: elevation / outlined
- square: disables border-radius

## Use
- Table container (required)
- Content panels and sections
- Bottom sheets
- Form containers

## Do Not
Apply Paper to every div — use only where visual separation is needed.
