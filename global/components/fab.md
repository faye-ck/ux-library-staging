# Floating Action Button

> **Category:** Inputs. Fixed-position primary action button for the most important action on a screen.

## Variants
- Regular (icon only)
- Extended (icon + label)

## Sizes
- Small
- Medium
- Large

## States
Default · Disabled

## Colors
Primary · Secondary

## Key Options
- Extended variant shows text label alongside icon
- Zoom transition animation on mount
- Speed dial — reveals multiple related actions
- Typically fixed bottom-right

## Use
- Primary create/add action
- Persistent global action
- Speed dial trigger

- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
