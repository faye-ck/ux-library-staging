# Avatar

> **Category:** Data Display. Visual representation of a user or entity — image, initials, or icon.

## Variants
- Circular (default)
- Square
- Rounded

## Sizes
- 24px · 32px · 40px (default) · 48px · 56px (custom)

## States
Image loaded · Fallback to initials · Fallback to icon

## Colors
Custom via bgcolor

## Key Options
- Image (src) — auto-fallback to initials on error
- Text initials — first letter of alt or explicit content
- Icon content
- AvatarGroup — stacked overlap with '+N' overflow indicator
- Max count in group

## Use
- User profile pictures
- Comment authors
- Team member lists
- Account menus

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
