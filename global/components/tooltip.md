# Tooltip

> **Category:** Data Display. Short contextual label appearing on hover or focus — explains an element without cluttering the UI.

## Variants
- Default (dark background)
- Custom (light or styled)
- With arrow

## Sizes
- Max-width 300px (configurable)

## States
Hidden · Visible (on hover / focus)

## Colors
Default dark · Customizable via sx

## Key Options
- 12 placement positions (top/bottom/left/right + start/end variants)
- Arrow — optional directional pointer
- Follow cursor — tracks mouse position
- Interactive — stays open when mouse enters it (WCAG compliant)
- Transition: Grow, Fade, Zoom

## Use
- Icon button labels
- Truncated text expansion
- Help hints
- Disabled element explanations

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
