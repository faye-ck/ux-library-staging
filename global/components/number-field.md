# Number Field

> **Category:** Inputs. Numeric input with stepper controls and optional min/max/step constraints.

## Variants
- Outlined
- Filled
- Standard

## Sizes
- Small
- Medium

## States
Default · Focused · Disabled · Error

## Colors
Primary · Secondary

## Key Options
- Min / max / step constraints
- Increment and decrement buttons (stepper)
- Helper text and error message
- Adornments — prefix unit, suffix label

## Use
- Quantity selectors
- Age / year fields
- Price or measurement input
- Settings with numeric values
