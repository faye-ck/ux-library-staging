# Speed Dial

> **Category:** Navigation. Expandable FAB revealing 2–6 related actions. Built on Fab.

## Directions
up (default) · down · left · right

## States
Closed · Open · Action hover · Action disabled

## Key Options
- direction: expansion direction
- open / onOpen / onClose: controlled
- hidden: for scroll-based visibility
- SpeedDialAction: tooltipTitle (required), icon

## Use
- Create actions (new task, event, message)
- Context-specific quick actions
- Mobile-primary workflows

## Do Not Use For
- Always-visible primary actions → use Button
- Destructive actions without confirmation
