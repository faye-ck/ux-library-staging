# Link

> **Category:** Navigation. Inline anchor element with consistent typography-based styling.

## Variants
- Inherit (default)
- Body1
- Body2
- Subtitle
- Caption

## Sizes
- Inherits from Typography variant

## States
Default · Hover · Visited

## Colors
Primary (default) · Inherit · Custom

## Key Options
- Underline: always (default), hover, none
- Typography variant — sets font size and weight
- External links — target='_blank' with rel='noopener'

## Use
- Inline text links
- Navigation links
- Action links in body text
- External references

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
