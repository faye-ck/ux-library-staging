# Snackbar

> **Category:** Feedback. Brief non-blocking notification that appears temporarily and auto-dismisses.

## Variants
- Basic message
- With action button
- Alert-style (with severity)

## Sizes
- Adapts to content

## States
Visible · Auto-dismissed · Manually dismissed

## Colors
Default (dark) · Alert severity via nested Alert

## Key Options
- Anchor position — 6 positions (top/bottom × left/center/right)
- Auto-hide duration — ms before auto-dismiss
- Action button — undo or dismiss CTA
- Transition: Grow (default), Fade, Slide
- Alert integration — success/warning/error/info styling

## Use
- Save/delete confirmation
- Undo action prompt
- Brief system notifications
- Non-critical feedback

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
