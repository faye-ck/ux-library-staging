# Checkbox

> **Category:** Inputs. Boolean selection control — supports checked, unchecked, and indeterminate states.

## Variants
- Default
- Custom icon
- Custom checked icon

## Sizes
- Small
- Medium

## States
Unchecked · Checked · Indeterminate · Disabled

## Colors
Primary · Secondary · Success · Error · Warning · Info

## Key Options
- Indeterminate — for parent/child group selection
- Label via FormControlLabel (placement: start / end / top / bottom)
- Custom icon and checked icon
- Group control via FormGroup

## Use
- Multiple selection from a list
- Parent/child selection groups
- Terms & conditions
- Boolean form fields

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
