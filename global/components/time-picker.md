# Time Picker

> **Category:** Utilities. Clock or digital interface for selecting a time value. (MUI X)

## Variants
- Desktop (default)
- Mobile (modal clock)
- Static (embedded)
- Digital (list-based)

## Sizes
- Default input width

## States
Default · Open · Disabled · Read-only · Error

## Colors
Primary (default)

## Key Options
- Views: hours, minutes, seconds
- 12-hour or 24-hour format
- AM/PM positioning (in clock face)
- Time step granularity (e.g., 5-minute steps)
- Digital clock threshold — list vs. clock view
- Disable specific times
- Orientation: portrait or landscape

## Use
- Appointment scheduling
- Meeting time selection
- Shift start/end time
- Deadline time input

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
