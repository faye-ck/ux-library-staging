# Menu

> **Category:** Navigation. Dropdown list of actions triggered by a button or icon — appears above the trigger.

## Variants
- Basic
- Icon menu
- Dense
- Selected menu
- Context menu (right-click)

## Sizes
- Default
- Dense (compact height)

## States
Open · Closed · Item selected · Item disabled

## Colors
Default (surface background)

## Key Options
- Dense — reduced item height and text
- Subheader — group labels within menu
- Dividers between item groups
- Icons — leading icons per item
- Selected item — visual highlight
- Context menu — triggered at cursor coordinates
- Nested / sub-menus

## Use
- Action dropdowns
- Context menus
- Account / profile menus
- More options (⋮) menus

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
