Stack

One-dimensional flex layout with consistent spacing between children. Use Stack when you need a row or column of elements spaced using the MUI spacing scale.

Utilities
MUI composition: Stack is a thin wrapper around CSS flexbox. The spacing prop maps directly to theme.spacing() — e.g., spacing={2} = 16px gap. No column spans or responsive breakpoints — use Grid for that.

Quick Reference

Direction

  • row (horizontal)
  • row-reverse
  • column (vertical, default)
  • column-reverse

Alignment (common)

  • alignItems: flex-start / center / flex-end / stretch
  • justifyContent: flex-start / center / space-between / flex-end

Key Props

PropPurpose
directionrow / column (default: column)
spacingGap between children (theme.spacing multiplier)
dividerInsert a Divider element between children
flexWrapAllow wrapping (nowrap by default)
useFlexGapUse CSS gap instead of margin (recommended for v7)

Common Use Cases

  • Form field stacking (column, spacing={2})
  • Button groups in a row (row, spacing={1})
  • Icon + label inline pairs (row, alignItems center)
  • Action bars with space-between layout

When NOT to Use

  • Two-dimensional layouts → use Grid
  • Irregular item sizes needing column spans → use Grid
Customization: Visual styling and theme overrides belong in the product's _theme.md. Component-level variants and usage rules specific to a product belong in _components.md.
