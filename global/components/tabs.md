# Tabs

> **Category:** Navigation. Tab bar for switching between related content views — horizontal or vertical.

## Variants
- Fixed (default)
- Full-width
- Scrollable (arrows on overflow)

## Sizes
- Default height
- Dense (icon-only)

## States
Default · Selected (active) · Disabled

## Colors
Primary (default indicator) · Secondary · Inherit

## Key Options
- Orientation: horizontal (default) or vertical
- Centered layout
- Scroll buttons: auto or always
- Icon only, icon + label, or label only
- Icon positions: top, start, end, bottom
- Indicator color
- Label wrapping for long names

## Use
- Content section switching
- View / mode selection
- Filter tabs
- Panel navigation

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
