# Divider

> **Category:** Data Display. Thin rule that separates content into clear sections or groups.

## Variants
- Full-width (default)
- Inset (indented)
- Middle (padded both sides)

## Sizes
- Inherits container width

## States
N/A

## Colors
Default (border color, customizable)

## Key Options
- Horizontal (default) or Vertical orientation
- Text content — label or chip in the center
- Text alignment: left, center, right
- Flex item mode — correct rendering inside flex containers

## Use
- List section breaks
- Form section separation
- Vertical dividers in toolbars
- Section headers with label

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
