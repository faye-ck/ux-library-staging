# Popover

> **Category:** Navigation. Floating panel anchored to an element — displays rich content in a layered overlay.

## Variants
- Anchor-based (default)
- Coordinate-based (virtual element)

## Sizes
- Adapts to content

## States
Open · Closed

## Colors
Default (surface background, with shadow)

## Key Options
- Anchor origin — alignment point on the trigger element
- Transform origin — alignment point on the popover
- Virtual element — position by coordinates
- Click-away and scroll-close behavior

## Use
- Rich tooltip content
- Inline settings panels
- Color pickers
- Floating edit forms

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
