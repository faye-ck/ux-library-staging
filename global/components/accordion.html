Accordion

Expandable/collapsible content sections for progressive disclosure.

Colors

Key Options

  • Custom expand icon
  • Default expanded on load
  • AccordionSummary — header with toggle
  • AccordionDetails — body content
  • AccordionActions — footer buttons
  • unmountOnExit — removes content from DOM when collapsed (performance)

Common Use Cases

  • FAQ sections
  • Settings panels
  • Collapsible content
  • Progressive disclosure
