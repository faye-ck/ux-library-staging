# Pagination

> **Category:** Navigation. Page navigation controls for splitting large datasets into discrete pages.

## Variants
- Text (default)
- Outlined
- Rounded

## Sizes
- Small
- Medium (default)
- Large

## States
Default · Current page (active) · Disabled

## Colors
Default · Primary · Secondary

## Key Options
- Shape: square (default) or rounded
- Show first/last buttons
- Hide prev/next buttons
- Sibling count — pages around current page
- Boundary count — pages at start/end
- Custom page icons
- Table pagination — rows-per-page + total count variant

## Use
- Table pagination
- Search results
- Article / post listing
- Data grid navigation

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
