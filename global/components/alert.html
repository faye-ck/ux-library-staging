Alert

Inline message communicating system status — success, info, warning, or error.

Feedback

Quick Reference

Variants

Standard (default)
Filled
Outlined

Sizes

Adapts to content

States

Static
Dismissible (with close button)

Colors

Success
Info
Warning
Error

Key Options

  • Severity — sets icon + color (success/info/warning/error)
  • AlertTitle — bold headline above message
  • Action — custom button or link inside alert
  • Custom icon or hidden icon
  • Dismissible — close button via onClose

Common Use Cases

  • Form feedback
  • System status banners
  • Validation messages
  • Inline warnings
Customization: Visual styling and theme overrides belong in the product's _theme.md. Component-level variants and usage rules specific to a product belong in _components.md.
