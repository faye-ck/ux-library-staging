# Badge

> **Category:** Data Display. Small label overlaid on an element — shows a count, status, or notification indicator.

## Variants
- Standard (number/text)
- Dot (minimal indicator)

## Sizes
- Adapts to content

## States
Visible · Invisible (hidden) · Zero (hidden unless showZero)

## Colors
Primary · Secondary · Success · Error · Info · Warning

## Key Options
- Anchor origin — corner position (top/bottom × left/right)
- Overlap mode — circular or rectangular base
- Max value — caps display (e.g., '99+')
- showZero — show badge when value is 0
- Invisible — toggle visibility programmatically

## Use
- Notification counts
- Unread message indicators
- Status dots on avatars
- Cart item counts

## Customization
- Styling and theme overrides: product `_theme.md`
