# Rating

> **Category:** Inputs. Star-based evaluation input. 1–5 scale by default.

## Sizes
Small · Medium · Large

## States
Default · Hover (preview) · Readonly · Disabled · Empty

## Key Options
- max: custom max value (default 5)
- precision: 0.5 for half-star
- readOnly: display-only
- getLabelText: accessible label per value (required)

## Use
- Feedback and satisfaction forms
- Content quality scores
- Display-only score indicators (readOnly)

## Do Not Use For
- Binary yes/no → use Switch or Checkbox
- Ranked ordering → use a ranked list
