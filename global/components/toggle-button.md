# Toggle Button

> **Category:** Inputs. Button-style selection control — one or multiple options can be active simultaneously.

## Variants
- Single selection (exclusive)
- Multiple selection

## Sizes
- Small
- Medium
- Large

## States
Default · Selected · Disabled

## Colors
Standard · Primary · Secondary · Error · Info · Success · Warning

## Key Options
- Full-width layout
- Horizontal (default) or Vertical orientation
- Icon only or icon + label
- Exclusive or multi-select mode

## Use
- View mode switching (list/grid)
- Text formatting (bold/italic/underline)
- Filter chips
- State selection

- Styling and theme overrides: product `_theme.md`
