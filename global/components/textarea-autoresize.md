# Textarea Autosize

> **Category:** Utilities. Textarea that automatically grows in height as content increases.

## Variants
- Standalone
- Integrated inside Text Field (multiline prop)

## Sizes
- minRows · maxRows configurable

## States
Default · Focused · Disabled

## Colors
Inherits from Text Field or custom

## Key Options
- minRows — minimum visible rows
- maxRows — caps height growth
- Full-width — stretch to container
- Used inside TextField with multiline prop

## Use
- Comments
- Message composition
- Notes fields
- Description inputs

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
