# Autocomplete

> **Category:** Inputs. Search-driven dropdown combining a text input with a filtered option list.

## Variants
- Outlined
- Filled
- Standard

## Sizes
- Small
- Medium

## States
Default · Focused · Disabled · Error · Loading

## Colors
Primary · Secondary

## Key Options
- Free solo — allows typed values not in the list
- Multiple selection — displays chips
- Grouped options — via subheader
- Filter-as-you-type or open-on-focus
- Custom option rendering

## Use
- Search fields
- Tag / label input
- Combobox with custom values
- Multi-select with chips

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
