# Button

> **Category:** Inputs. Triggers an action or event — the primary interactive element in any UI.

## Variants
- Contained (filled)
- Outlined
- Text (ghost)

## Sizes
- Small
- Medium
- Large

## States
Default · Hover · Focused · Disabled · Loading

## Colors
Primary · Secondary · Success · Error · Warning · Info

## Key Options
- Start icon / end icon
- Icon-only button (pair with Tooltip)
- Full-width layout
- Link behavior (renders as anchor)
- File upload button
- Loading state with spinner

## Use
- Primary actions
- Secondary actions
- Destructive actions
- Form submit

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
