# Text Field

> **Category:** Inputs. General-purpose text input — the standard form field for user data entry.

## Variants
- Outlined (default)
- Filled
- Standard

## Sizes
- Small
- Medium

## States
Default · Focused · Disabled · Error · Required · Read-only

## Colors
Primary · Secondary · Success · Warning · Error

## Key Options
- Multiline with auto-resize (Textarea Autosize)
- Input adornments — prefix icons, suffix units
- Helper text and character count
- Floating label
- Input types: text, password, email, search
- Full-width layout

## Use
- Form data entry
- Search fields
- Comments / descriptions
- Email, password, name fields

- Styling and theme overrides: product `_theme.md`
