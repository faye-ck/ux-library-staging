Grid

12-column responsive grid for two-dimensional page layouts. Use Grid when content must respond to breakpoints with different column spans — cards, dashboards, form columns.

Utilities
MUI composition: Grid v2 (MUI v7). Uses CSS flexbox with a 12-column base. Two roles: container (the row wrapper) and item (each column cell). Column spans are set per breakpoint using xs, sm, md, lg, xl props.

Key Props

PropPurpose
containerMarks the Grid as a row container
itemMarks a Grid child as a column cell
xs / sm / md / lg / xlColumn span at each breakpoint (1–12 or "auto")
spacingGap between items (theme.spacing multiplier)
columnsOverride total column count (default: 12)
directionrow / column

Common Use Cases

  • Dashboard card layouts (3 or 4 columns desktop, 1 mobile)
  • Two-column forms (side-by-side fields on md+)
  • Article with sidebar layout
  • Product / feature listing grids

When NOT to Use

  • Single-direction layouts → use Stack
  • Simple flex alignment → use Box with sx
Customization: Visual styling and theme overrides belong in the product's _theme.md. Component-level variants and usage rules specific to a product belong in _components.md.
