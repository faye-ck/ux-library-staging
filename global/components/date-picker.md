# Date Picker

> **Category:** Utilities. Calendar-based date selection input — desktop and mobile variants. (MUI X)

## Variants
- Desktop (default)
- Mobile (modal calendar)
- Static (embedded, no input)

## Sizes
- Default input width (customizable)

## States
Default · Open · Disabled · Read-only · Error

## Colors
Primary (default)

## Key Options
- Views: day, month, year
- Orientation: portrait or landscape
- Display week numbers
- Show days outside current month
- Min date / max date constraints
- Reduce animations

## Use
- Date of birth
- Event scheduling
- Deadline selection
- Reporting date range

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
