# Breadcrumbs

> **Category:** Navigation. Shows the user's current location in the site hierarchy as a navigable path.

## Variants
- Default
- With icons
- Collapsed (ellipsis with expand)

## Sizes
- Adapts to content

## States
Default · Active (last item, non-interactive)

## Colors
Default · Primary links

## Key Options
- Custom separator — text ('›', '/', '-') or icon
- maxItems — collapses middle items to ellipsis
- Leading / trailing icons per item
- Last item as non-interactive Typography
- Underline control on links

## Use
- Page location indicator
- Navigation trail
- Back navigation context
- Hierarchy display

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
