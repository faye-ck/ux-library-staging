# List

> **Category:** Data Display. Vertical stack of rows — supports icons, avatars, text, and nested sub-lists.

## Variants
- Text only
- Icon + text
- Avatar + text
- Two-line
- Nested

## Sizes
- Default
- Dense (compact row height)

## States
Default · Selected · Disabled

## Colors
Primary (selected state)

## Key Options
- Dense — reduces row height for compact layouts
- Subheader — sticky section label
- Secondary action — icon or button at row end
- ListItemButton — full-row clickable area
- Nested lists — indented sub-items
- Dividers between items

## Use
- Navigation menus
- Contact/user lists
- Settings items
- Data listings with actions

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
