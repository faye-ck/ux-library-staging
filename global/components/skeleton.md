# Skeleton

> **Category:** Feedback. Content placeholder during loading. Prefer over spinner for lists and content-heavy views.

## Variants
text · rectangular · rounded · circular

## Animation
pulse (default) · wave · false (reduced motion)

## Key Options
- width / height: required for rectangular/circular
- Match skeleton dimensions to real content

## Use
- List rows loading (text variant)
- Card images (rectangular)
- Avatars (circular)
- Multi-line text blocks

## Do Not Use For
- Non-data-dependent interactive elements
- Page-level operations → use LinearProgress
