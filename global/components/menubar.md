# Menu Bar

> **Category:** Navigation. Horizontal application-level menu bar with nested menus — desktop-style navigation.

## Variants
- Default
- With keyboard shortcut hints
- With checkbox/radio items

## Sizes
- Full-width by default

## States
Default · Open · Item focused · Item disabled

## Colors
Default (surface background)

## Key Options
- Shortcut hints — keyboard shortcut labels
- Checkbox items — toggleable menu entries
- Radio group items — exclusive selection
- Icon menu items
- Group labels via subheaders

## Use
- Application-level navigation
- Desktop-style toolbar menus
- Feature menus with keyboard shortcuts

## Customization
- Styling and theme overrides: product `_theme.md`
- Component-level usage rules: product `_components.md`
