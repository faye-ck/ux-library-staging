# Typography (component)

> **Category:** Data Display. Renders text using MUI's type scale variants. See Foundations → Typography for the token scale.

## Variants
h1 · h2 · h3 · h4 · h5 · h6 · subtitle1 · subtitle2 · body1 · body2 · caption · overline · button

## Colors
inherit · primary · secondary · disabled · error

## Key Options
- variant: type scale token
- component: rendered HTML element (decoupled from variant)
- noWrap: single-line with ellipsis
- gutterBottom: adds bottom margin
- paragraph: adds paragraph spacing

## Use
- Page/section headings: h1–h4
- Card titles: subtitle1; descriptions: body2
- Helper text: caption
- Labels/tags: overline
- Body copy: body1

## Reference
Foundations → Typography for font specs and scale.
