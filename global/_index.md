## Design system foundation

Global is built on MUI v7 (Material UI). The `mui-design` skill encodes all Global section rules for AI-assisted documentation.

For products using Ant Design, see `ant-design/_index.md`. Use `/ant-design-spec` to generate parallel antd mapping pages for existing specs — the skill never modifies original spec files.
