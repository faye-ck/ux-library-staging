Template Layout — Mobile Web

Delta only. Base spec: feature.html · Anything not listed here is identical to the base spec.

Note on this delta: The Figma frame provided (node 1911:200130, "Responsive spec") does not contain a dedicated mobile phone design. It shows the desktop User Portal at two additional viewport breakpoints: 1320px and 2558px (1920+). No phone-specific UI (single-column layout, bottom navigation, touch-optimised tap targets) is present.

This delta documents the responsive layout behaviour of the User Portal across these breakpoints. A true phone-width mobile web design (≤ 768px) appears to be absent from the Figma file. confirm with designer whether a phone design is planned.
