Approval Flow — Feature

Step 2 of 3 in the Template Creation wizard (Layout → Approvals → Preview). Template creators configure whether approval is required and select or create the approval flow to attach to the template.

Context: This screen is accessed by template creators (admin role) via "Create New Template". Figma breadcrumb: Create New Template / Approval Flow Setup. Confirm whether this spec belongs under admin-portal or user-portal.

Core Flows
Flow 1 — Select an existing approval flow
  1. Template creator lands on Approvals step (stepper: Layout ✓, Approvals active)
  2. Left panel shows "Default approval flow" section; toggle "Approval required" is ON by default
  3. Creator clicks "Select approval flow" dropdown → list expands with existing templates
  4. Creator selects a template (e.g., "Finance department") → dropdown closes, selection shown
  5. Creator clicks "Save as Draft" or "Publish" to proceed Confirm Publish enabled state
Flow 2 — Approval not required
  1. Creator toggles "Approval required" switch to OFF
  2. Select dropdown and pencil icon disappear
  3. Empty state shows: "This workflow will simply collect responses"
  4. Creator proceeds to "Publish" Confirm
Flow 3 — Create a new approval flow
  1. Creator clicks pencil/edit icon next to "Select approval flow" dropdown
  2. Background dims; "Create approval flow" modal opens (1400px wide)
  3. Modal breadcrumb: Create New Template / Create approval flow
  4. Title* field shows auto-name: "New approval flow 1"
  5. Creator fills optional Description field
  6. "Allow self-approval" is pre-checked — creator may uncheck
  7. Step 1 row shown; right panel "Step 1" settings open with Fixed tab active
  8. Creator selects approver type (User / Department / Assigned group) and searches for approvers
  9. Added approvers shown in list: avatar + name + email + × remove
  10. Creator clicks "Add approval step" → Step 2 added (type defaults to "Final approver")
  11. Creator configures Step 2 in right panel
  12. Creator clicks "Save" → modal closes; new AF saved and auto-selected in main dropdown
Flow 4 — Change step approver type
  1. Creator clicks step type label dropdown (e.g., "Approver ▼") on a step row
  2. Dropdown shows available types — at minimum: "Approver", "Final approver" Confirm full list
  3. Creator selects new type → step row and right panel update
Flow 5 — Reorder or remove steps
  1. Each step row shows: duplicate icon, delete (trash) icon, collapse chevron
  2. Duplicate: clones step below current Inferred
  3. Delete: removes step; confirmation if last step Confirm
  4. Collapse chevron: collapses step row to show header only
