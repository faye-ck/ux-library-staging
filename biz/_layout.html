Feature Layouts

Seat Booking

Full-screen Office Map view. Left menu + top bar shell. Map fills the content area.

Right panel (399px) opens on seat selection — shows seat detail and booking form. Slides in-flow; not a modal.

Map overlays: date navigation component + on-map action buttons. Shadow: drop-shadow(0px 4px 20px rgba(0,0,0,0.15)).

Seat Booking Confirmation

Confirmation shown after a seat is booked. Booking summary: seat, date/time, user.

INFERRED MUI Dialog (maxWidth="sm") or right panel overlay. Confirm / Cancel actions.

Space Booking Confirmation

Same pattern as Seat Booking Confirmation but for meeting space / room bookings.

INFERRED MUI Dialog (maxWidth="sm") or right panel overlay.

My Bookings

List view of current and upcoming bookings. Standard portal shell (left menu + top bar).

Content: paginated table or card list. Pagination: MUI TablePagination.

Empty state: Empty State component (Illustration variant) when no bookings exist.
