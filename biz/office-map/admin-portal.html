3 — Map Editor Improvements

Replaces pin-point only spaces with polygon drawing tools; improves creation flow intuition.

Figma →

Phase 1 Implemented

Onboarding: 4-step (dev: 3 steps — polygon not done).

Space Creation: Assets in accordion. Pin → pin-point; polygon → draw area (Phase 1: polygon hidden). Color = status color. Alignment toolbar on multi-select. Anchor point editor.

Seat Creation: Drag to place. Toolbar: count (H/V) · rotation · color (#808080, fixed opacity). Multi-select → alignment (6-point + distributed). Label shows on hover/select only. Toggle label visibility.

Map Area Creation: New zone for heatmap/analytics. Draw polygon → fill info → save. Default: #C8F7F5 @ 50% opacity.

Saving: Toast on success · Save disabled if no changes · Confirmation on close with unsaved changes.
