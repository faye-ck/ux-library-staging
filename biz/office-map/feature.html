Office Map — Feature

User portal baseline. Role sections inside for different permission levels.

draft Updated 2026-05-01

F01 — Seat Map Navigation

Groups relevant UI actions into three screen areas: Header, Map navigation, and Left menu.

Header Navigation

  • Company Logo — refreshes current page on click.
  • Search and filter — opens F08 Search and filter from map.

Map Navigation

  • Add New button — dropdown: Space booking (F02) · Seat booking (F03) · Bulk seat booking (F04).
  • My Bookings — opens F10.
  • User Profile — shows Avatar, Name, Clock-in status. Panel tabs: Time Logs · My Status (F06) · Proxied Users (F07, only if proxied users exist). Full-width panel on mobile.

Left Menu Navigation

Collapsed by default. Menu items in order: Seat Map · Users · Space · Scheduler · Inventory · Locker · Analytics · Manual.

F02 — Space Booking

Allows booking a space within 48 hours or for consecutive full days.

I. Space Details

Shows space picture, name, description, type, and capacity. Timeline picker: 0:00–23:59+1, 15 min slots. Click or drag to pre-fill booking screen.

II. Booking Screen

  • By time: Timeline + time picker (2-way sync, available slots only).
  • By days: Calendar picker. Max 7 consecutive days. Up to 6 rolling months. Reserves 0:00–23:59.
  • Participants: Roles: Organizer · Participant. Guests cannot be organizers.
  • Title/Remark: Optional. Title max 100 chars; Remark max 320 chars.
  • Private booking: Lock icon shown to non-participants; hides count and blocks click.

III. Confirmed Booking Screen

Shows booking details, participant list (sorted: Creator → Organizers → Participants → Guests), and remark. Buttons: Edit · Duplicate · Make another booking.

IV. Booking Details

Action buttons evolve based on state:

StateButtons
Before startEdit · Duplicate · Cancel
10 mins before startStart Event · Cancel
Active/ongoingExtend Time · Exit Space

F02-EXT — Space Booking Time Extension

Consistent extend-time options across user portal, mobile app, and space signage.

  • 15 mins
  • 30 mins
  • 60 mins — hidden if less than extendable duration
  • Until next event (xx mins) — shows balance; capped at admin's Max Extension Duration; "max." prefix if actual > max
  • Custom — text field; restricted to indicated max

If booking is restricted to business hours, end of business hours is factored into max duration.

F03 — Seat Booking

Allows booking a seat for single or multiple dates.

I. Seat Details

Shows seat name, description, type (Individual / Group), and capacity. Date selector with calendar picker, 2-way sync with map. Users list sorted by clock-in state then time. Buttons: Clock-in · Change Seat · Clock-out · New Booking.

II. Booking Screen

Calendar picker (multi-select, max 15 days). Time slots from admin settings (A01); auto-selects if only 1 slot. Optional remark field.

III. Confirmed Booking Screen

Shows seat name, date/time slot list in two columns. Buttons: Edit · Make another booking.

IV. User Profile

Avatar · Name · Role · Department · Personal number · Clock-in status · Participation status · Contact buttons (Email, Teams/Google). Custom status + last updated. Timeline view 0:00–23:59, 15 min slots.

F04 — Bulk Seat Booking

Books multiple seats for multiple users within the same day and time slot.

  • Calendar picker + time slots from A01.
  • Participants list: first unassigned user auto-selected; next auto-selected after each assignment.
  • Validation: unassigned user error · duplicate individual seat error.
  • Confirmed screen: participants sorted alphabetically by seat name. Buttons: Edit · Duplicate · Make another.

F06 — Custom Status

User sets a custom status message, auto-clear timer, and external link on their profile.

  • Avatar: Upload .jpg/.png; auto-crop 1:1; max 2MB.
  • Message textarea: Max 280 chars. Live character count.
  • Auto clear: Never (default) · Today · 1 hour · 4 hours · This week · Custom date/time.
  • External link: Title (max 65 chars) + URL. Error shown for invalid URL.
  • Clear Status: clears textarea, resets to Never.
  • Save Status: saves and updates last-updated timestamp on profile.

F07 — Proxy User Management

Profile panel tab for managing clock-in status of proxied users.

I. Proxied User List

Bulk and individual actions: Clock-in + select seat · Clock-in without seat · Working remotely · Working offsite · Clock-out. Repeated statuses from bulk action ignored. All changes update time logs; triggers "Status updated" email.

II. Seat Assignment Screen

Title: "Proxied User Seat Assignment". First clocked-out user auto-selected; next after each assignment. Optional remark. Confirm triggers email to all users.

III. Confirmed Assignment Screen

Participants sorted by seat name. Remark shown if present.

F08 — Search and Filter from Map

Search, filter, and locate users from the map.

I. Search and Filter

Text field: "Search name, email address, space or seat". Filter pill dropdown: Department or Custom Group (one at a time).

II. Results Screen

Total count + tab navigation (when 2+ result types). Listing: Avatar · Name · Role · Department · Personal number · Clock-in status · Second status (Participation > Custom). Contact buttons. On mobile: floats at 50% height, drag to 90%.

III. User Profile

Refer to F03 IV. Map highlights selected user.

F09 — Custom Group

Create and manage custom groups for reuse in search and bookings.

Group List

Shows total count. Each group: title · members count (hover tooltip) · context action button · Edit · Delete (confirmation prompt).

FeatureButton
Seat MapPin icon — Locate in map
SchedulerSwap icon — Replace to calendar

Create / Edit Group

Required group title. Members list with drag-and-drop ordering (order number auto-updates). Create/Save disabled if title or member list empty.

Group Details

Members: Avatar · Name · Clock-in status · Custom status · Contact buttons. Context action + Edit + Delete.

F10 — My Bookings

Manage upcoming bookings across Space, Seat, and Inventory.

Tab Navigation

Tabs: Space · Seat · Inventory. Badge shows total count; hidden when empty.

Space Bookings

  • Actions: Duplicate · Edit · Cancel.
  • Ongoing: "Ongoing" divider. Actions: Extend Time · End Meeting.
  • Starting Soon: "Starting Soon" divider. Actions: Start Event · Cancel.

Seat Bookings

Seat name · Date · Slots (comma-separated). Actions: Edit · Cancel.

Inventory Bookings

Borrowing: Take inventory · Edit · Return item. Bookings: Borrow (after start) · Edit · Cancel. Dedicated items: not counted in badge. Take inventory · Return item.
