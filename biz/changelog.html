Biz — Changelog

Content history for Biz. Each entry records what was added or updated, and when.

v0.5.0 2026-05-01 Minor
Added
Locker — Feature, Mobile Web, Admin Portal
Full spec from ClickUp. Feature: locker dashboard (4 summary grids), office map search/grid/panel, transfer and booking flows, locker booking during seat booking, My Bookings tab. Mobile: auto/manual start, pick-up confirmation, QR scanning flows (group, transfer, booking, dedicated). Admin Portal: user portal settings, email notifications, Create Locker Group (general settings, layout, grid preview, cell merging, non-lockers), group details, edit locker, CSV column changes, search tables, In Use Lockers report, map editor placement.
Added
Office Map — Feature (F01–F10)
Full user portal spec from ClickUp. Covers seat map navigation (F01), space booking + time extension (F02/F02-EXT), seat booking (F03), bulk seat booking (F04), custom status (F06), proxy user management (F07), search and filter from map (F08), custom group (F09), and My Bookings (F10).
Added
Office Map — Admin Portal
Dynamic time slot settings (with-time-slots and full-day-only setup tables) and map editor improvements (Phase 1: space creation, seat creation, map area creation, keyboard shortcuts; Next Phase items listed).
Added
Analytics — Admin Portal
Seat/space analytics dashboard: global filters (time period, location), sub-menu (Space Utilization, Seat Utilization, Check-in by Department), time dimension toggle table, stacked bar charts, grid heat map with color coding (5 utilization bands), Business Hours setting, and Check-in by Department.
Added
Reception — Admin Portal + Receptionist Portal
Security Card Management spec. Admin Portal (sections 1–3): enable/disable feature, create/edit cards (fields table, CSV bulk update), manage unpaired/paired cards (pairing queue, bulk actions, card status reference table, search). Receptionist Portal (section 4): guest check-in (skip card with In: prefix), guest check-out (skip card with Lost/Damaged/Other reason options, Out: prefix, bulk mixed check-out).
Added
Inventory — Feature
Bulk Booking/Borrowing spec: switching to bulk mode (right panel changes, eligibility checks, mobile behaviour), evidence upload flow, timeline view (multi-date item-level selection), My Bookings (bulk booking and borrowing/returning), and Phase 2 Workflows integration (upcoming).
Changed
All content pages — Sources moved to bottom
Moved "Source:" paragraph from the page header area to a dedicated "Sources" section at the end of each page, formatted as a bullet list. Applied to 9 pages: Office Map (feature, admin-portal), Analytics (admin-portal), Reception (admin-portal, receptionist-portal), Inventory (feature), Locker (feature, mobile-web, admin-portal).
Changed
Stub pages — CSS and status badge standardized
Updated 15 stub HTML pages to use the standard feature page template: max-width 760px, updated md-h3 styling (11px/600/0.07em), added ul/li rules, spec-table styles, and status-badge classes. Added "draft" status badge with last-updated date to all stub pages missing it.

Status upgraded to stable.
Changed

Expanded component catalog: Button variants (10 Figma types), Chip types (Round / Square / Filled Info, 32 Filled Info variants), Modal (added Scrolling Long Dialog, Action Footer mobile), Select (added Filter Select and Filter Select Transparent), custom icon catalog (6 categories, 70+ named icons), Empty State illustrations (10 context types), Date Navigation (Single + Month), Avatar component set, Profile/User List, Timeline Slot sub-component. All CONFIRM/INFERRED flags resolved. Status upgraded to stable.

Layout — Figma keys + confirmed token values
Added Figma import keys for Portal Template, Left Menu, Top Bar, and Panel Header. Confirmed Left Menu hover state tokens. Added Content Area / Panel Header section. Removed INFERRED flag from breakpoints (all values confirmed). Status upgraded to stable.

Added _figma-api.md entry to the Config Files table. Status upgraded to stable.
