Import Patterns

// Component SET (has variants)
const set = await figma.importComponentSetByKeyAsync('KEY');
const variant = set.findChild(c => c.type === 'COMPONENT' &&
  c.variantProperties?.State === 'Default' && c.variantProperties?.Size === 'Medium');
const inst = variant.createInstance();

// Single component (no variants)
const comp = await figma.importComponentByKeyAsync('KEY');
const inst = comp.createInstance();

// Set props — TEXT and BOOLEAN use exact '#uid' key; VARIANT use plain names
inst.setProperties({ 'Title#34279:0': 'Save', 'Front icon#34306:52': false });

Design Token Import Keys

Use figma.variables.importVariableByKeyAsync(key) for color and spacing tokens. Use figma.importStyleByKeyAsync(key) for text styles. Import only what each script needs — importing all at once causes timeouts.
