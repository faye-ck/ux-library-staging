Biz — Theme

MUI color tokens and typography for the Biz User Portal (web). Source: Figma library -Library- Biz User Portal v2.

✓ stable Last updated: 2026-05-01  ·  Framework: MUI v5+

Framework

MUI v5+ (Material UI). Theme delivered via createTheme(). CSS variables mode (cssVariables: true) generates --mui-palette-* vars alongside the Figma custom property names shown below.
