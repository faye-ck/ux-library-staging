Inventory — Feature

User portal baseline for inventory requests and borrowing.

draft Updated 2026-05-01

Bulk Booking/Borrowing

Select multiple items in one booking or borrowing session. Handles varied item configs — evidence, Workflows, and different max durations. Max duration: 30 days.

1 — Switching to Bulk Mode

From booking form: New Booking → "Bulk Booking" · Borrow Now → "Bulk Borrowing".

Right Panel

  • Title changes. Item-specific fields removed.
  • Item list: sorted by add time · remove via "x" · empty = "No item selected" (Confirm disabled).
  • Return date field shown for Bulk Borrowing.

Left Section — Eligibility Checks

Checkboxes on grid/list; timeline slots as checkboxes. Only eligible items selectable — checks: bookable without approval · available in range · max duration within range · lease duration within range.

After date/time update: Unavailable items turn red. Error alert with "Remove all unavailable items" button. Confirm disabled until resolved.

Mobile: Panel closes on bulk mode; floating back button. Confirmation screen shows items sorted alphabetically.

2 — Evidence Required

"Evidence is required for some items" message. Camera icon → opens camera. Upload icon → take photo or choose file. Confirm disabled until all evidence uploaded.

3 — Timeline View

Multi-date: clicking any slot of an item selects all its slots across dates (item-level selection). Dates beyond 7-day view: start date auto-shown as first column (3-day on mobile).

4 — My Bookings

Bulk Booking

Title "Bulk booking (#)". Actions by status: Edit · Borrow Now (when ready) · Cancel. Item list expand/collapse (sorted alphabetically). Edit = same creation flow. Cancel = confirmation prompt.

Bulk Borrowing/Returning

Title "Bulk borrowing (#)" + "Return All Items". Item list with checkboxes and Return icons. "Return (#)" for multi-return. Item details: Take Inventory + Return Item. Remaining time shown when returning.

Phase 2 — Workflows Upcoming

  • Items requiring permission bookable but cannot mix with non-permission items.
  • All items must share the same Workflows template (one Workflow per bulk booking).
  • Approval message shown. Edit = resets approval process.
  • My Bookings: Workflows status before item list. Additional filter for Workflow templates.
